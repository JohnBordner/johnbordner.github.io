NFL Predictor

Web application built using Spring and Thymeleaf. Users can log in, predict games, track their win-loss record, and compare their score with others!

The app uses Tank01's API to pull NFL schedules and results, tracking your record.

Technologies: Spring (Java), Thymeleaf, MariaDB (MySQL), Tank01 Football API
